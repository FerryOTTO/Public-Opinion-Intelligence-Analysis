问题诊断

问题: 浏览器正在请求 /src/main.ts，但应该请求编译后的 JS 文件。

可能原因:

  1. GitHub Pages 使用了错误的源分支
  2. 构建过程没有正确替换 src="/src/main.ts"
  3. 浏览器缓存了旧的 HTML 文件
  4. GitHub Actions 工作流程失败

立即解决方案

  1. 清除浏览器缓存: Ctrl+Shift+Delete 或使用隐私模式
  2. 检查 GitHub Pages 设置:
    • 前往仓库 Settings → Pages
    • Source 应该显示 "GitHub Actions"
    • 不是 "Deploy from a branch"
  3. 检查 GitHub Actions:
    • 前往 Actions 标签页
    • 查看最新的 "Deploy to GitHub Pages" 工作流程
    • 确保它成功完成
